## 📌 Course Description

This course covers:
- Solution of **nonlinear equations**
- **Interpolation** and approximation
- **Numerical differentiation** and integration
- Solution of **linear algebraic equations**
- Solution of **ordinary differential equations**
- Solution of **partial differential equations**

> It provides foundational knowledge for **numerical analysis** used in scientific computing and engineering applications.

## 📐 Key Formulas Quick Reference

### Unit 1 — Root Finding

| Method | Formula |
|---|---|
| **Bisection** | `x_new = (a + b) / 2` |
| **False Position** | `x = (a·f(b) - b·f(a)) / (f(b) - f(a))` |
| **Newton-Raphson** | `x_(n+1) = x_n - f(x_n) / f'(x_n)` |
| **Fixed Point** | `x_(n+1) = g(x_n)` |

### Unit 2 — Interpolation

| Method | Formula |
|---|---|
| **Lagrange** | `P(x) = Σ [f(x_i) · L_i(x)]` |
| **Newton Forward** | `P(x) = f(x_0) + sΔf(x_0) + s(s-1)/2! · Δ²f(x_0) + ...` |
| **Least Squares (Linear)** | `y = a + bx` where `b = (nΣxy - ΣxΣy) / (nΣx² - (Σx)²)` |

### Unit 3 — Integration

| Method | Formula |
|---|---|
| **Trapezoidal** | `h/2 · [f(x_0) + 2Σf(x_i) + f(x_n)]` |
| **Simpson's 1/3** | `h/3 · [f(x_0) + 4Σf(odd) + 2Σf(even) + f(x_n)]` |
| **Simpson's 3/8** | `3h/8 · [f(x_0) + 3Σf(x_1,x_4...) + 3Σf(x_2,x_5...) + 2Σf(x_3,x_6...) + f(x_n)]` |

### Unit 5 — ODEs

| Method | Formula |
|---|---|
| **Euler's** | `y_(n+1) = y_n + h·f(x_n, y_n)` |
| **RK4** | `y_(n+1) = y_n + (1/6)(k1 + 2k2 + 2k3 + k4)` |
| **Modified Euler** | `y_(n+1) = y_n + h/2 · [f(x_n,y_n) + f(x_(n+1), y*_(n+1))]` |

## ⚠️ Common Errors to Avoid

| Error Type | Description |
|---|---|
| **Round-off Error** | Caused by finite digit representation in computers |
| **Truncation Error** | Caused by approximating infinite processes (e.g., Taylor series) |
| **Absolute Error** | `|True Value - Approximate Value|` |
| **Relative Error** | `|Error| / |True Value|` |
| **Percentage Error** | `Relative Error × 100%` |
