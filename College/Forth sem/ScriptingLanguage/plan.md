### MySQL — Common Commands

```sql
-- Connect and select database
USE database_name;

-- Create table
CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- CRUD Operations
INSERT INTO students (name, email) VALUES ('Ram', 'ram@example.com');
SELECT * FROM students WHERE id = 1;
UPDATE students SET email = 'new@example.com' WHERE id = 1;
DELETE FROM students WHERE id = 1;

-- Joins
SELECT s.name, c.course_name
FROM students s
INNER JOIN courses c ON s.course_id = c.id;

-- Aggregate
SELECT COUNT(*), AVG(marks), SUM(marks) FROM results;
```

### AJAX — Basic Structure

```javascript
// Vanilla JS AJAX
const xhr = new XMLHttpRequest();
xhr.open('POST', 'process.php', true);
xhr.setRequestHeader('Content-Type', 'application/x-www-form-urlencoded');
xhr.onreadystatechange = function () {
    if (xhr.readyState === 4 && xhr.status === 200) {
        document.getElementById('result').innerHTML = xhr.responseText;
    }
};
xhr.send('name=Ram&email=ram@example.com');

// jQuery AJAX
$.ajax({
    url: 'process.php',
    type: 'POST',
    data: { name: 'Ram', email: 'ram@example.com' },
    success: function(response) {
        $('#result').html(response);
    }
});
```

### PHP OOP — Class Structure

```php
<?php
class Student {
    // Properties
    public $name;
    private $marks;

    // Constructor
    public function __construct($name, $marks) {
        $this->name = $name;
        $this->marks = $marks;
    }

    // Method
    public function getGrade() {
        if ($this->marks >= 90) return 'A+';
        elseif ($this->marks >= 80) return 'A';
        else return 'B';
    }
}

// Inheritance
class GraduateStudent extends Student {
    public $thesis;

    public function __construct($name, $marks, $thesis) {
        parent::__construct($name, $marks);
        $this->thesis = $thesis;
    }
}

// Usage
$s = new Student("Ram", 85);
echo $s->getGrade(); // A
?>
```

### JS Form Validation — RegEx Patterns

| Field | RegEx Pattern |
|---|---|
| **Name** | `/^[a-zA-Z\s]{3,50}$/` |
| **Email** | `/^[^\s@]+@[^\s@]+\.[^\s@]+$/` |
| **Phone (Nepal)** | `/^(98\|97)\d{8}$/` |
| **Password** | `/^(?=.*[A-Z])(?=.*\d).{8,}$/` |
| **URL** | `/^(https?:\/\/)?([\w-]+\.)+[\w]{2,}$/` |
